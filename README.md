# AM.SALES VPN — прототип

Десктоп-приложение на **C++ / Qt 6 (QML)**. Пока это **прототип**: красивый
интерфейс (зелёно-чёрно-белая палитра, цифровой звёздный фон, планета с
орбитой), кнопка включения, которая запускает/останавливает **zapret**, и
панель сканирования устройств в локальной сети.

> ⚠️ VPN-часть пока **не подключена** — это следующий этап. Сейчас приложение
> управляет только обходом DPI (zapret).

---

## Что умеет прототип

- 🟢 **Кнопка включения** — запускает `zapret` (с запросом прав администратора)
  и останавливает его. Планета и кнопка подсвечиваются, когда защита активна.
- 🛰 **Анимации** — звёздное поле, вращающийся спутник на орбите, пульсация.
- 📡 **Сканер сети** — находит устройства в твоей локальной сети (IP + MAC),
  подсвечивает твой ПК.

## Как собрать — самый простой путь

На этой машине всё уже подготовлено:

- **Visual Studio 2022** (компилятор C++, CMake, Ninja) — установлена в `F:\vs`.
- **Qt 6.8.1** (MSVC 2022 64-bit) — установлен локально в папку `Qt/` внутри
  проекта (через `aqtinstall`, без аккаунта Qt).

### Собрать в один клик

1. Запусти **`build.bat`** (двойной клик).
2. Скрипт сам подтянет Qt из папки проекта, окружение компилятора Visual Studio,
   сконфигурирует, скомпилирует и запустит приложение.

> `build.bat` ищет Qt в таком порядке: локальная папка `Qt/6.8.1/msvc2022_64`
> → переменная `QT_DIR` → стандартные пути `C:\Qt`. Если Qt установлен в другое
> место — задай `set QT_DIR=путь` перед запуском.

### Установка Qt заново (если папку Qt удалили)

Qt ставится одной командой через aqtinstall (нужен Python, он есть):

```bat
python -m pip install --user aqtinstall
python -m aqt install-qt windows desktop 6.8.1 win64_msvc2022_64 --outputdir "%CD%\Qt"
```

## Как собрать вручную (если хочется понимать процесс)

```bat
:: 1. Сконфигурировать (создаётся папка build)
cmake -S . -B build -DCMAKE_PREFIX_PATH=C:\Qt\6.8.1\msvc2022_64

:: 2. Скомпилировать
cmake --build build --config Release

:: 3. Скопировать библиотеки Qt рядом с .exe (чтобы запускался без Qt в PATH)
C:\Qt\6.8.1\msvc2022_64\bin\windeployqt.exe --qmldir qml build\Release\AmSalesVPN.exe

:: 4. Запустить
build\Release\AmSalesVPN.exe
```

## Структура проекта

```
AmSalesVPN/
├── CMakeLists.txt          ← как собирать проект
├── build.bat               ← сборка в один клик
├── README.md               ← этот файл
├── assets/
│   └── AmSalesLogo.png      ← логотип

│   └── NetworkScanner.*     ← сканер локальной сети
└── qml/                     ← интерфейс (анимации, вёрстка)
    ├── Main.qml             ← главный экран
    ├── StarField.qml        ← звёздный фон
    ├── PlanetOrbit.qml      ← планета с орбитой
    └── GlowButton.qml       ← кнопка включения
```

## Настройка пути к zapret

По умолчанию приложение запускает:
```
C:/Users/danec/Desktop/zapret-universal/general (ALT3).bat
```
Если твой zapret лежит в другом месте — поменяй путь в файле
[src/ZapretController.cpp](src/ZapretController.cpp) в конструкторе
(`m_batPath = ...`), затем пересобери.
